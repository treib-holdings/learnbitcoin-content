CPU 挖矿是使用通用计算机处理器而非专用硬件来挖比特币。它是 2009-2010 年挖比特币的唯一方式，但在比特币主网上已经经济性灭绝了十多年。

历史进程：

- **2009-2010 年：** CPU 挖矿。任何有笔记本电脑的人都能找到区块。[Satoshi](/glossary/satoshi-nakamoto) 和 [Hal Finney](/glossary/hal-finneys-running-bitcoin) 就是这样挖矿的。
- **2010 年底：** GPU 挖矿开始。显卡在哈希运算上比 CPU 快大约 50-100 倍。CPU 矿工无法竞争。
- **2013 年：** [ASIC](/glossary/asic-application-specific-integrated-circuit) 到来。GPU 矿工在几个月内变得不具竞争力。
- **2014 年起：** ASIC 专属时代。CPU 挖矿产生的算力占网络算力的 0.000000001%，如果有的话。

CPU 挖矿为什么仍在文化中占有一席（微小）之地：

- **教育价值。** 在测试网或回归测试网上运行 CPU 矿工是许多开发者最初学习哈希实际含义的方式。
- **彩票硬件。** 多年来，少数人因在廉价硬件上独挖而意外赢得了主网区块——一个 USB 矿机、一个家用 GPU，偶尔是一个 CPU。概率天文数字般地低，但彩票特性吸引了一些爱好者。
- **一些山寨币仍然偏好 CPU 挖矿**（如 Monero 的 RandomX）。那里的经济学不同。在比特币上，CPU 挖矿纯粹是情怀。

如果你在比特币主网上以 100% 运行 CPU 一整年，你预期的区块发现数是一个小数点后有很多零的数字。你可能会加热你的办公室，吓到你的电力供应商，而且永远不会找到一个区块。[算力](/glossary/hash-rate)的数学不偏爱业余者。

2026 年真正有效的挖矿方式参见[矿机](/glossary/mining-rig)。
